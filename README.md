<h2>🚀 English Version</h2>

<h3>Overview</h3>
<p>LDM Assistant Web is a professional tool built on the <b>Web Bluetooth API</b>. It allows engineers, designers, and surveyors to collect laser distance measurements directly via a web browser. No app installation is required, and it supports offline use via PWA technology.</p>
<p>⚠️ <b>Note:</b> This tool supports <b>Precaster OEM/ODM based</b> laser distance meters only.</p>

<h3>📱 System Compatibility</h3>
<ul>
  <li><b>Android</b>: Google Chrome (Recommended).</li>
  <li><b>Windows / macOS / ChromeOS</b>: Google Chrome or Microsoft Edge.</li>
  <li><b>iOS (iPhone/iPad)</b>:
    <ul>
      <li><i>Note</i>: Safari does NOT support Web Bluetooth.</li>
      <li><i>Solution</i>: Download the <b>"Bluefy"</b> or <b>"WebBLE"</b> browser from the App Store to use this tool.</li>
    </ul>
  </li>
</ul>

<h3>📥 How to Install (PWA)</h3>
<ol>
  <li><b>Android</b>: Open in Chrome -> Tap 3-dots menu -> Select <b>"Install App"</b> or <b>"Add to Home Screen"</b>.</li>
  <li><b>iOS (via Bluefy)</b>: Tap the Share button -> Select <b>"Add to Home Screen"</b>.</li>
</ol>

<h3>🛠️ Operation Guide</h3>
<ol>
  <li><b>Connect</b>: Turn on LDM Bluetooth. Click <b>[Connect LDM]</b> and select your device (usually named "Laser" or "LDM").</li>
  <li><b>Measure</b>:
    <ul>
      <li><b>Manual</b>: Trigger on the device; data appears in the list automatically.</li>
      <li><b>Remote</b>: Click the <b>[Measure]</b> button in the web UI.</li>
    </ul>
  </li>
  <li><b>Manage</b>: Use <b>[Clear]</b> to reset the list or <b>[Export / Share]</b> to save the data as a CSV file.</li>
</ol>

<a name="traditional-chinese"></a>
<h2>🚀 繁體中文版</h2>

<h3>工具簡介</h3>
<p>LDM 網頁助手是一款基於 <b>Web Bluetooth API</b> 開發的專業工具，專為工程測量、室內設計與巡檢人員設計。無需下載安裝包，直接透過瀏覽器即可與雷射測距儀連線，並支援 PWA 技術實現離線作業。</p>
<p>⚠️ <b>注意：</b>本工具僅支援基於 <b>Precaster OEM/ODM 協議</b>之雷射測距儀。</p>

<h3>📱 系統相容性</h3>
<ul>
  <li><b>Android 系統</b>：請使用 <b>Google Chrome</b> 瀏覽器（最推薦）。</li>
  <li><b>Windows / macOS</b>：請使用 <b>Google Chrome</b> 或 <b>Microsoft Edge</b>。</li>
  <li><b>iOS (iPhone/iPad)</b>：
    <ul>
      <li><i>注意</i>：原生 Safari 不支援藍牙連線。</li>
      <li><i>解決方案</i>：請至 App Store 下載 <b>"Bluefy"</b> 或 <b>"WebBLE"</b> 瀏覽器開啟此網頁。</li>
    </ul>
  </li>
</ul>

<h3>📥 如何安裝到桌面 (PWA 指南)</h3>
<ol>
  <li><b>Android (Chrome)</b>：點擊選單（三個點） -> 選擇 <b>「安裝應用程式」</b> 或 <b>「加入主畫面」</b>。</li>
  <li><b>iOS (Bluefy)</b>：點擊分享按鈕 -> 選擇 <b>「加入主畫面」</b>。</li>
</ol>

<h3>🛠️ 操作指南</h3>
<ol>
  <li><b>連線設備</b>：開啟測距儀藍牙，點擊網頁上的 <b>[Connect LDM]</b> 並選擇您的設備。</li>
  <li><b>進行測量</b>：
    <ul>
      <li><b>手動測量</b>：直接操作硬體，數據將即時顯示於網頁清單中。</li>
      <li><b>遠端測量</b>：點擊網頁上的 <b>[Measure]</b> 按鈕可遙控設備測量。</li>
    </ul>
  </li>
  <li><b>數據管理</b>：點擊 <b>[Clear]</b> 可清空紀錄；點擊 <b>[Export / Share]</b> 可將清單匯出為 CSV 檔（Excel 可開啟）。</li>
</ol>

<h3>⚠️ Troubleshooting | 疑難排解</h3>
<ul>
  <li><b>Bluetooth not found? / 找不到藍牙？</b> Ensure your OS Bluetooth is ON and the web address starts with <code>https://</code>.<br />
    請確認系統藍牙已開啟，且網址必須以 <code>https://</code> 開頭。</li>
  <li><b>Auto-Input into Excel? / 自動輸入到 Excel？</b> This Web version is for <b>data logging</b>. For "Barcode Scanner Mode" (typing directly into cells), please use our <b>Windows Desktop Version</b>.<br />
    網頁版主要用於**數據記錄**。若需要像條碼槍一樣直接打字進入 Excel 儲存格，請改用我們的 **Windows 桌面專業版**。</li>
</ul>
